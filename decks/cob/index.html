Cogent Co-Browser

Secure, Shared Browsing as a service

Proposal By Fractal Technologies

Terminologies

Cogent The company owning the CoBrowsing Solution
CoBrowser The Software under Discussion
Company A company which is a Client of CoBrowser, and has integrated it on one or more of their website. Cogent itself is a Client of CoBrowser
Support Person In house support person who'd sit on this tool to provide support for a Company
User The end user using Company's website and receiving support using CoBrowser
Company Extension Browser extension downloaded automatically at Company's end at the time of software setup
User Extension Browser extension which can be downloaded by power users

Vision

Company should be able to buy and setup CoBrowser as a service for their website(s) without human intervention

Company should be able to easily secure various input fields on their website

Company's support person should be able to co-browse only the Company's website(s) with a User, and also Chat with them

No User Extension download required at User's End initially. Tab switching enabled using popups.

User Extension download can be used to disable these popups if required for regular Users.

CoBrowsing Session would be recorded at Support Provider's side and stored with Cogent

Cogent should be able to provide support for CoBrowser using this very technology

Company Website Plugin

A <script src="cobrowser.plugin.js"> which will be inserted in the Company Website's <head> by their developers to enable Cobrowsing and Chat based support. They will get this text from Co-Browser App.

This plugin will manage synchronization of browsing on User's end. This will also apply security filters before sending data.

Api Server

This server will manage all the data and associated business rules for CoBrowser. This consists of all the data of registered Companies, their Support Persons, chat logs, access to videos of previous support sessions, security filters, payments etc.

Socket Server

This server will manage real time transfer of browsing activity between User and Support Person. It will also enable chat between User and Support Person.

App

A Company will need to create an account with CoBrowser App, and add credentials of their support persons.

Support Persons will automatically download the Support Extension when they login using their credentials from a new browser(which doesn't have the extension installed).

Support Extension

Browser Extension in which Support Person logs in. This browser extension manages synchronization of browsing on Support Person's end.

This extension will also control what all actions can the support person take on the User's browser, and will send cobrowsing session data to the recording server.

Recording Server

Server which records and stores media stream picked up by the Support Extension.

User Extension

Browser extension to smoothen up tab switching. Only needed for power users.

Development Plan

Phase 1

Company Website Plugin (Extendable Event Pipeline, with security filters, Chat)
Api Server (Company Registration, Support Management, Security Filters, Chat Logs, Session Logs)
Socket Server (Horizontally Scalable Channels, which allow pipelining data b/w User and Support Person)
App (UI for Company Registration, Login, Support management, Chat logs, Session Logs, Website Plugin)
Support Extension (Extendable Event Pipeline, Chat)

Phase 2

Recording Server (Save incoming video streams in files via scokets, notify api when a recording is done)
Support Extension (Start screen-recording at the start of a co-browsing session, Send the stream to recording server, and optionally to socket server)
Company Website Plugin (Optional preview of what the support guy is seeing will be shown to the user)
Api Server (Support Session Videos Module)

Phase 3

Api Server (Automated Payment Integration)
User Extension (User Extension for power Users)

Timelines

Phase 1 2 months
Phase 2 1 months
Phase 3 1 months

Handoff Process

1 month into the project, FractalTech will bring a talented software developer onboard, who'd work on CoBrowser's development. This Software developer will later be FractalTech's dedicated consultant to Cogent for CoBrowser App
